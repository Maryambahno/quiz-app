# Projekt - Historiequiz

Detta projekt är en del av en kursuppgift där vi arbetar i grupp enligt Scrum-ramverket.
Syftet med uppgiften är att träna på samarbete, sprintar och arbetsprocessen i Scrum, inte att bygga en avancerad applikation.
Applikationen byggs som en enkel webbsida med HTML, CSS och JavaScript.

## Projektbeskrivning
Vi bygger en enkel quiz-app med lätta historiska frågor.
Användaren ska kunna:
1. Se en fråga
2. Välja ett svar
3. Gå vidare till nästa fråga
4. Få ett resultat i slutet

## Roller i projektet
. Product Owner: Ansvarar för vad som ska byggas och prioritering av uppgifter
. Scrum Master: Ansvarar för att Scrum-processen följs och att möten hålls
. Utvecklare: Alla gruppmedlemmar som arbetar med kod och uppgifter

Rollerna används enligt kursens instruktioner och Scrum Master roteras vid behov.

## Arbetsmetod (Scrum)
Projektet är uppdelat i sprintar, där varje sprint är en vecka.
Varje sprint innehåller:
. Sprint Planning
. Daily Scrum
. Sprint Review
. Sprint Retrospective

Alla uppgifter hanteras som GitHub Issues och organiseras i en GitHub Project Board (Kanban-tavla).
