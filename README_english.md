# Project - History Quiz

This project is part of a school assignment where we work in a group using the Scrum framework.  
The purpose of the assignment is to practice collaboration, sprints, and the Scrum workflow, rather than building a complex application.  
The application is developed as a simple webpage using HTML, CSS and JavaScript.

## Project Description
We are building a simple quiz app in swedish with basic history questions.  
The user should be able to:
1. View a question  
2. Select an answer  
3. Move on to the next question  
4. Receive a final result at the end  

## Project Roles
- Product Owner: Responsible for defining what should be built and prioritizing tasks  
- Scrum Master: Responsible for ensuring the Scrum process is followed and that meetings are held  
- Developers: All group members who work on coding and tasks  

The roles are used according to the assignment instructions, and the Scrum Master role rotates when needed.

## Working Method (Scrum)
The project is divided into sprints, where each sprint lasts one week.  
Each sprint includes:
- Sprint Planning  
- Daily Scrum  
- Sprint Review  
- Sprint Retrospective  

All tasks are managed as GitHub Issues and organized in a GitHub Project Board (Kanban board).
